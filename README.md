# EduTech-Task12-Predictive-Analytics

## Tools Used
Python, Pandas, Scikit-learn, Matplotlib, Seaborn, Joblib

## Dataset
Marketing Linear Regression Dataset (Kaggle)
Link: https://www.kaggle.com/datasets/fayejavad/marketing-linear-multiple-regression

## Steps Performed
1. Loaded marketing dataset (YouTube, Facebook, Newspaper vs Sales)
2. Train-Test Split (80-20)
3. Linear Regression model trained
4. Model evaluation using RMSE and R2 Score
5. Actual vs Predicted plot
6. Feature Importance visualization
7. Prediction report saved as CSV
8. Model saved as .pkl file

## Results
- RMSE shows average prediction error
- R2 Score shows how well model fits data
- YouTube has highest impact on sales

## Interview Answers
**Regression vs Classification?**
Regression predicts continuous values jaise sales amount, price, temperature.
Classification predicts categories jaise spam/not spam, yes/no, 0/1.
Linear Regression ek regression technique hai jo continuous output deta hai.

**What is RMSE?**
RMSE (Root Mean Square Error) model ki prediction accuracy measure karta hai.
RMSE = actual aur predicted values ke difference ka square root.
RMSE jitna kam hoga, model utna accurate hoga.
